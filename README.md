# Thememux

Rust implementation of termux-theme-changer. This tool helps to change termux theme from the command line.

## Awailable themes

- Base16 default dark
- Base16 default light
- Base16 greenscreen dark
- Dracula
- Gnometerm new
- Gruvbox dark
- Gruvbox light
- Material
- Solarized dark
- Solarized light
- Tokyonight dark
- Tokyonight day
- Tomorrow night
- Ubuntu
- Zenburn
